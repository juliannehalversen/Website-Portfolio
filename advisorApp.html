Advisor App

Built for Alliance Benefit Group Rocky Mountain using Angular 8 and Ionic 5

The Details

I got instructed the project of rebuilding a mobile native app for our advisors without any previous knowledge of Angular or Ionic, and very limited knowledge of REST APIs, authentication, and how to connect it all. So, I bought several Udemy courses and dove into all the documentation I could find. I built this app on my own and learned the necessary technologies to build it, along with all their quirks. However, the most important thing I learned was how to learn and teach myself.

Once logged in, this app consists of main dashboard with an overview their employees, funds, models, and overall data. They can then select individual plans to see participation, distributions, contributions, etc. From there they can drill down on an individual employee to see their contribution information along with previous account activity. There is much more that went into this app but I feel like this is the core functionality. This app is living in a private company repo but I'd be happy to provide demo login credentials.

Building the App

I worked with many different REST endpoints to get the data to my app. It was built with Angular, styled with Ionic, and deployed with Cordova(and Ionic).

What I learned:

  • Angular
  • Ionic
  • REST API Process
  • Authentication Methods(authentication tokens/cookies)
  • Working/filtering large amounts of JSON
  • Deploying to different mobile devices
  • How to learn when faced with work that is completely new to me!
